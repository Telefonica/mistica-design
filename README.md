## Install

```
yarn add
```

## Local preview

```
yarn start
```

## CLI deployment

1. Assign to project (if you already have `mistica-design` asing that)
2. Create `mistica-design` project

### Pre-deploy

```
yarn predeploy
```

This will create a preview of the deployment in the vercel project.

### Deploy to production

```
yarn deploy
```
